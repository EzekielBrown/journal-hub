Week 9


CSS lengths can be divided into two types. Absolute lengths and relative lengths. There are 6 absolute lengths and 9 relative lengths. The absolute length units are fixed and a length expressed in any of these will appear as exactly that size. As screen sizes vary so much, using absolute length units on a screen is not recommended. A relative length unit specifies a length one relative to another. A relative length unit scales better across different rendering mediums.

Absolute Length

cm - centimeters
mm - millimeters
in - inches (1 = 2.54cm)
px - pixels (1 = 1/96th of 1in)
pt - points (1 = 1/72 of 1in)
pc - picas (1 = 12pt)

Relative Length

em - One em is a distance equal to the text size
This code will set the span to 15px
div { font-size: 30px; } span { font-size: 0.5em; }

ex - x-height of the current font OR one-half of one em
This code will set the text in the span to half the size vertically
div { font-size: 30px; } span { font-size: 1ex; }

ch - ch is the width of the number 0 in the font.
This code sets the font size to 3times the width of 0
body { font-size:16px; } div { font-size: 3ch; }

rem - The rem unit sets the font-size relative to the browsers base font-size, and will not inherit from its parents.
This code translate to times 2 of the browser font size
html { font-size:16px; } div { font-size: 2rem; }

vw - 1vw = 1% of viewport width.
This code scales with the size of the browser and will change when the browser to effected horizontally
h1 { font-size: 20vw; }

vh - hundredths of the viewport height.
This code scales with the size of the browser and will change when the browser to effected vertically
h1 { font-size: 20vh; }

vmin - hundredths of whichever is smaller, the viewport width or height.
This code, scales with the size of the browser and stops at a certain size and will not getting bigger
h1 { font-size: 20vh; }

vmax - hundredths of whichever is larger, the viewport width or height.
This code scales with the size of the browser and stops at a certain size and will not get smaller
h1 { font-size: 15vmax; }

% - percentage relative to the parent element
This code increased the div font 150% of 16px. So the new font size will be 24px.
body { font-size:16px; } div { font-size: 150%; }
